a_gw_wannier90_band.dat is the gw-band data interpolated by Wannier90;
a_pbe_band is the band data of pbe by VASP;
a_pbe_kpt is the k-point coordinate corresponding to pbe-band converted into the abscissa coordinate.
b_bse_dielectric contains the imaginary part of the dielectric function obtained by GW+BSE;
b_ipa_vasprun.xml is the GW+RPA output file obtained by VASP, which contains the dielectric function.
